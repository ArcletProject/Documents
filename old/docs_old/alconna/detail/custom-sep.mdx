## 自定义分隔符

~~NLP~~ (大雾)

`Alconna` 不强制 shell-like 的指令, 即支持自定义分隔符, 所以以下命令是可以的

```python
from arclet.alconna import Alconna, Args, AnyParam

alc = Alconna(
    command="{name}今天",
    main_args=Args["item":AnyParam],
    action=lambda x: x.split("和"),
    separator='吃'
)
```

:::tip PS.

虽然说 `CommandNode` 支持空字符串作为分隔符, 但是主命令的分隔符不能是空字符串, 否则解析会出错

如果你想在主命令用空字符串作为分隔符的话, 为什么不试试第一种写法呢？
