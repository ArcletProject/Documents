## 快捷命令

在`v0.6.4`中, `Alconna` 加入了方法 `shortcut`，用来为命令添加快捷指令。

顾名思义, 就是把一长串命令用一个特别的字符串来表示, 以便更方便的使用。

```python
from arclet.alconna import Alconna, Subcommand, Option, Args

github = Alconna(
    command="!github",
    options=[
        Subcommand(
            "repo",
            options=[
                Option(
                    "--issues",
                    Args["issue":str:"issues_id"],
                    help_text="获取仓库的issue, 可以指定issue id"
                ),
                Option(
                    "--pulls",
                    Args["pull":str:"pulls_id"],
                    help_text="获取仓库的pull, 可以指定pull id"
                ),
                Option(
                    "--branch",
                    Args["branch":str:"main"],
                    help_text="配合 codeview 使用, 指定分支"
                ),
                Option(
                    "--codeview",
                    Args["path":str:"file_path"],
                    help_text="预览 path"
                ),
            ],
            args=Args["repo":str:"repo_name"],
            help_text="查看github仓库的信息"
        ),
    ],
)
```
