得益于`Alconna`的完善性, 在 v0.6 版本中, 我们加入了命令行功能.

## Commandline

在通过`pip`安装/更新 `arclet-alconna` 后, 你可以尝试在本地终端输入

```powershell
PS C:\Users\Alconna> alconna
```

import {alconna} from "@site/src/components/commandlines.js";


<TerminalApp  prompt={"C:\\Users\\Alconna> "} lines={alconna} height={360}/>

## create指令

该指令用来辅助入门者快速创建一个 `Alconna` 命令.

import {alconna_create} from "@site/src/components/commandlines.js";


<TerminalApp  prompt={"C:\\Users\\Alconna> "} lines={alconna_create} height={420}/>


## analysis指令

该指令用来将一个普通的命令解析为以字典形式表示的命令结构.


## using指令

该指令需要配合`create`来使用, 它用来执行在cli中创建好的`Alconna`指令, 并输出对应的解析结果.

## help指令

该指令用来快速查看Api文档.
