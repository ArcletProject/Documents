得益于`Alconna`的完善性, 在 v0.6 版本中, 我们加入了命令行功能.

## Commandline

在通过`pip`安装/更新 `arclet-alconna` 后, 你可以尝试在本地终端输入

```powershell
PS C:\Users\Administrator> alconna
```
