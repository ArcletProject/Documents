## 减少 Option 的使用

利用 `K` 与 `O` 标识符，我们可以在 Args 中模拟出一个 option：

:::info

`K` 与 `O` 的作用请看这里: [参数标识符](../basic/alconna-args.mdx#标识符控制)

:::


```python
from arclet.alconna import Alconna, Args

alc = Alconna(
    "cut_img",
    Args["--width;O|K":int:1280, "--height;O|K":int:720]
)
```
