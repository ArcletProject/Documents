## 减少 Option 的使用

利用 `@` 与 `?` 前缀, 我们可以在 Args 中模拟出一个option:

:::info

`@` 与 `?`的作用请看这里: [`Args`](../basic/alconna-args.mdx)

:::


```python
from arclet.alconna import Alconna, Args
alc = Alconna(
    "cut_img",
    Args["@?--width":int:1280, "@?--height":int:720]
)
```
