## 自定义分隔符

~~NLP~~ (大雾)

`Alconna` 不强制 shell-like 的指令, 即支持自定义分隔符, 所以以下命令是可以的

```python
alc = Alconna(
    command=f"{AnyStr}今天",
    main_args=Args["item":AnyParam],
    actions=lambda x: x.split("和")
).separate("吃")
```
