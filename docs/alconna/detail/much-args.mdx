## 选项的多参数

`Args` 可以填入不止一个参数，所以以下命令是可以成功解析的

```python
from arclet.alconna import Alconna, Args, Option

cal = Alconna(
    command="Cal",
    main_args=Args["num_a":int, "num_b":int],
    options=[
        Option("-sum"),
        Option("-div"),
        Option(
            "--round", 
            Args["decimal":int]
        ),
    ],
)
```
